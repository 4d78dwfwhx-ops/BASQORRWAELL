# 🚗 منصة باسكور - Basqor

منصة متكاملة لبيع قطع غيار السيارات الأصلية

## 📋 المتطلبات

- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- Apache/Nginx مع mod_rewrite
- SSL Certificate (HTTPS)

## 🚀 التثبيت

1. ارفع الملفات إلى الخادم
2. افتح `https://yourdomain.com/install.php`
3. اتبع خطوات التثبيت
4. احذف ملف `install.php` بعد الانتهاء
